# End of Match Awards

At the end of every race, three awards are presented to recognize
different strengths.

## 🏆 Champion

**Awarded to:** Player with the highest overall score.

### Score Calculation

Points are accumulated throughout the race.

Suggested scoring:

-   Correct character: **+1 point**
-   Completed phrase: **+50 bonus**
-   First to complete a phrase: **+20 bonus** *(optional)*
-   Mistyped character: **0 points** *(or a small penalty if desired)*

**Winner**

``` text
Champion = Player with the highest total score
```

**Tie breakers**

1.  More phrases completed
2.  Higher typing speed
3.  Earlier finish time

------------------------------------------------------------------------

## 🎯 Sharpshooter

**Awarded to:** Player who completed the most phrases correctly.

This measures consistency rather than speed.

### Calculation

``` text
Sharpshooter = Highest number of correctly completed phrases
```

**Tie breakers**

1.  Higher accuracy
2.  Faster average phrase completion time

------------------------------------------------------------------------

## ⚡ Speed Demon

**Awarded to:** Player with the highest typing speed.

### Calculation

``` text
WPM = (Total Correct Characters / 5) / Minutes Spent Typing
```

**Example**

``` text
850 correct characters
Typing time: 4 minutes

WPM = (850 / 5) / 4 = 42.5
```

**Tie breakers**

1.  Higher accuracy
2.  More phrases completed
